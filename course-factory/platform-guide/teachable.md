# 🎒 Teachable / Thinkific / Podia

Three nearly-identical platforms for self-paced course hosting. Pick the one with a UI you find least annoying — the feature differences rarely matter for a first course.

## Pick this if

- Self-paced course (no live cohort element)
- You want fast setup — usually under 8 hours from blank to ready-to-sell
- Modest community needs (basic Q&A is enough)
- Modest email needs (you'll use ConvertKit / Beehiiv separately)

## Don't pick this if

- You're running a cohort — you'll fight the platform constantly
- You want native email + sales funnel + course in one tool — go Kajabi
- You want custom branding to the level where students forget they're on a 3rd-party platform

## Pricing (verify current — these change)

| Platform | Free tier | Starter | Pro |
|---|---|---|---|
| Teachable | Yes (with fees) | ~$39/mo | ~$119/mo |
| Thinkific | Yes (1 course) | ~$49/mo | ~$99/mo |
| Podia | No | ~$33/mo | ~$75/mo |

Most charge transaction fees on the lowest tier. Higher tiers remove them.

## What they do well

- Course-hosting with video, drip, prerequisites, certificates
- Sales page + checkout
- Student progress tracking
- Quiz / assessment tools
- Affiliate program (Pro tiers)
- Mobile-friendly student app

## What they do mediocre

- Email marketing — basic. Use a real ESP alongside.
- Community — basic discussion threads. Real community needs Circle / Mighty / Discord.
- Live calls — none native. Use Zoom + a calendar tool.
- Customization — themes are dated. You can override CSS but it's a fight.

## Recommended setup

1. Sign up. Pick the lowest tier — upgrade later only if you hit a real limit.
2. Create the course shell. Modules first, lessons within. Don't worry about content yet.
3. Set up the sales page. Use the platform's templates — don't custom-design.
4. Connect Stripe.
5. Set up drip if relevant (release one module per week vs all at once).
6. Add captions to every video before launch. Auto-gen, then proofread.
7. Connect ConvertKit / Beehiiv / your ESP via Zapier or native integration for email.
8. **Run a test purchase yourself** before going live.

## Migrating later

All three let you export student data + course content. Migrating to Kajabi or self-hosted is painful but possible. Plan to stay on one of these for at least 2 years after first launch.
