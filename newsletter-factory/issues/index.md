# 📰 Issues

The issue archive. Each issue lives in its own folder under `issues/<NNN-slug>/`:

| File | What it is |
|---|---|
| `index.md` | Issue overview — subject lines, opening hooks, send metadata |
| `draft.md` | The issue itself, paste-ready |

## Folder convention

```
issues/
  001-launch/
    index.md
    draft.md
  002-<slug>/
    ...
```
