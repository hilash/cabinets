# 📐 ConvertKit / Kit

> The platform built for creators who sell products. Recently rebranded as Kit. Strongest automation, tagging, and segmentation. Less of a "newsletter platform", more of an "audience operating system".

## Pick this if

- You sell (or will sell) digital products, courses, coaching, books
- You want surgical control over who sees what (segments, tags, sequences)
- You run launches (welcome flows, evergreen funnels, cart-abandonment, etc.)
- You don't care about being on a discovery network

## Don't pick this if

- You just want to write and ship — the configuration surface is overkill
- You're not selling anything and don't expect to

## Pricing

- Free: up to 10,000 subscribers, but limited features (no automation, no commerce)
- Creator: ~$15–25/mo for under 1,000 subs, scales with list size
- Creator Pro: ~$25–50/mo, adds advanced features

## Strengths

- **Tags + segments.** Anyone can be tagged based on what link they clicked, what form they filled, what product they bought. Then you target.
- **Sequences.** True drip automation — welcome series, course launches, cart sequences.
- **Commerce.** Sell digital products, paid newsletters, tip jars, all native.
- **Sparkloop integration.** Best paid-recommendation network (you pay for high-quality subs from other newsletters).
- **Forms / landing pages.** Native, decent.
- **Creator Network.** ConvertKit's recent answer to Substack — recommendations from other Kit users.

## Weaknesses

- Configuration surface is huge. Easy to spend a week setting up before sending anything.
- Templates are functional but plain — visual polish requires custom work.
- The "newsletter" surface (web archive, public profile) is weaker than Substack/Ghost.

## Recommended setup

1. Sign up. Connect a domain you own (or use ConvertKit's subdomain).
2. Build ONE form (the main subscribe form). Don't build five forms before sending an issue.
3. Build ONE sequence (the welcome — just 3 emails: welcome, your best past piece, an ask question).
4. Tag your subscribers by acquisition source.
5. Send weekly. Use broadcasts for issues, sequences for evergreen flows.
6. Add commerce only when you have something specific to sell.

## ConvertKit vs Beehiiv vs Substack

If you sell things → ConvertKit. If you want network growth → Substack. If you want growth tools without the commerce focus → Beehiiv. Many writers eventually run two: ConvertKit for the audience+commerce, plus Substack/Beehiiv for the public-facing newsletter. That's overkill for the first 1,000 subs.
