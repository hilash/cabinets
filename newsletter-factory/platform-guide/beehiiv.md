# 🐝 Beehiiv

> Built by ex-Morning Brew operators. Best-in-class growth tooling and monetization, with a Substack-tier free plan.

## Pick this if

- You're serious about growth tools (referral program, ad network, segmentation)
- You want to monetize through ads (Beehiiv has a built-in ad marketplace) AND/OR paid subs
- You want a free tier that doesn't feel crippled
- You want a custom domain on the free plan

## Don't pick this if

- You want a community / discovery network like Substack's (Beehiiv has a smaller "Boost" network but it's pay-to-play)
- You're a creator with products to sell — ConvertKit's automation is more powerful

## Pricing

- Free: up to 2,500 subscribers, includes custom domain, basic features
- Scale: ~$39/mo
- Max: ~$99/mo
- Enterprise tiers above that
- 0% revenue fee on paid subs (you keep more than Substack)

## Strengths

- **Referral program.** The best in the business. Tier-based rewards, automated.
- **Ad network.** If you don't want paid subs but still want revenue, Beehiiv's ad network places sponsored content in your issues for you.
- **Segmentation + tagging.** Send different content to different cohorts.
- **Boosts.** Pay other newsletters to recommend you (it works — measurable subs per dollar).
- **Web archive + SEO.** Issue archive ranks in Google. Substack does this too but Beehiiv's is more flexible.
- **Polls and surveys.** Native, embedded in issues.

## Weaknesses

- No equivalent to Substack's free recommendation network — growth is more pay-to-play.
- UI more complex than Substack — slight learning curve.
- Smaller community of writers (this is changing fast).

## Recommended setup

1. Sign up. Add custom domain.
2. Set up the referral program day one — tiered rewards (5 referrals = bonus issue, 25 = sticker, 100 = AMA, etc.).
3. Configure segmentation with at least one tag per acquisition source (so you know which channels work).
4. Enable Boosts — modest budget ($50/mo for early-stage) to grow.
5. Apply to the ad network once you have 1,000+ subs and want revenue without going paid.
6. Set up the welcome sequence — Beehiiv's automation is decent for this.
