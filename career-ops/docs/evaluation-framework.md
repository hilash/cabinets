# The 6-Block Evaluation Framework

Every job offer is evaluated across six blocks before any application effort is spent. This is the quality gate that prevents spray-and-pray behavior.

## Block A — Role Summary (20%)

| Score | Criteria |
| --- | --- |
| 5.0 | Perfect scope, clear growth path, strong team, exciting mission |
| 4.0 | Good scope with minor unknowns, reasonable growth, solid team |
| 3.0 | Scope is vague or too narrow, growth uncertain, team is question mark |
| 2.0 | Red flags: team of one, scope soup, "wearing many hats" |
| 1.0 | Fundamentally misaligned with career goals |

## Block B — CV Match Analysis (25%)

| Score | Skills Overlap |
| --- | --- |
| 5.0 | 90%+ hard requirements met, strong keyword alignment |
| 4.0 | 75-90% met, minor gaps addressable in cover letter |
| 3.0 | 50-75% met, significant gaps in key areas |
| 2.0 | Below 50%, would need to stretch considerably |
| 1.0 | Fundamental mismatch — different domain/stack entirely |

## Block C — Level Strategy (15%)

| Score | Criteria |
| --- | --- |
| 5.0 | Perfect level match or strategic step-up with realistic path |
| 4.0 | Right level, some stretch in one dimension (scope, team size) |
| 3.0 | Slightly over or under-leveled — workable but not ideal |
| 2.0 | Clearly over-leveled (risk of failure) or under-leveled (bored in 6 months) |
| 1.0 | Level mismatch would undermine career trajectory |

## Block D — Compensation Research (15%)

| Score | Criteria |
| --- | --- |
| 5.0 | Above market, excellent equity, total comp exceeds target |
| 4.0 | At market, competitive equity, within target range |
| 3.0 | Below market but negotiable, or low equity component |
| 2.0 | Significantly below market, limited flexibility signals |
| 1.0 | Not competitive for the level and location |

## Block E — Personalization Tips (10%)

| Score | Criteria |
| --- | --- |
| 5.0 | Unique "unfair advantage" for this specific role (domain, connection, timing) |
| 4.0 | Strong angle — clear proof points map directly to their needs |
| 3.0 | Generic match — nothing specific differentiates you from other candidates |
| 2.0 | Weak angle — hard to articulate why you specifically |
| 1.0 | Cultural or values misalignment |

## Block F — Interview Prep (15%)

| Score | Criteria |
| --- | --- |
| 5.0 | Strong STAR+R stories mapped, format is comfortable, intel available |
| 4.0 | Good story coverage, minor prep gaps, interview format is familiar |
| 3.0 | Some stories map, unknown interview format, limited company intel |
| 2.0 | Weak story coverage for likely questions, unfamiliar format |
| 1.0 | No relevant stories, completely unknown company or interview process |

## Weighted Score Calculation

```
Score = (A × 0.20) + (B × 0.25) + (C × 0.15) + (D × 0.15) + (E × 0.10) + (F × 0.15)
```

## Decision Thresholds

| Score Range | Action |
| --- | --- |
| **4.5+** | Top priority. Apply immediately. Fast-track CV tailoring. |
| **4.0-4.4** | Strong apply. Standard tailoring and application. |
| **3.5-3.9** | Consider. Only with warm referral or thin pipeline. |
| **3.0-3.4** | Weak. Skip unless strategically valuable. |
| **Below 3.0** | Skip. Move to Withdrawn. |
