## Architecture: User Layer vs System Layer

Adapted from the original career-ops data contract:

| Layer | What's In It | Who Changes It |
| --- | --- | --- |
| **User Layer** | Profile, CV, proof points, tracker data, stories | You |
| **System Layer** | Agents, jobs, evaluation framework, portal configs | System updates |

System updates never overwrite your User Layer data. Your profile, CV, and pipeline are yours.
