# Play

Where the fun work lives.

## What lives here

- `habits/` — what you're tracking, why, and whether it's working.
- `dnd/` — kids' campaign: world, NPCs, session prep, recaps.
- `projects/` — side projects, tinkering, half-finished ideas.
- `hobbies/` — anything you do for joy. Reading list. Climbing log. Chess openings.
- `briefs/` — weekly review.

## Team

- **Habit Architect** — designs the tracker *with* you. Rebuilds dashboards. Tells you honestly when a habit is a lie.
- **DnD DM** — worldbuilder, NPC wrangler, session prep, post-session recap. Keeps the kids' campaign alive between sessions.

## Weekly rhythm

- **Sunday** — Habit Architect reviews last week's habits. What stuck. What didn't. Why.
- **Friday night or whenever you play** — DnD DM ships the pre-session brief.
- **After session** — DM writes the recap from your scratch notes. Updates NPCs + world state.

## Habits template (minimal)

```
## [Habit name]
- Target: [specific, measurable]
- Why: [your reason, in your own words]
- Signal you'll watch: [how you'll know it's working]
- Running since: YYYY-MM-DD
- Log: [link to CSV or table]
```

## DnD seed structure

```
dnd/
  campaign.md          ← premise, tone, current arc
  world/
    map.png
    regions/
    factions/
  npcs/                ← one file per NPC with voice, goals, secrets
  pcs/                 ← one file per player character
  sessions/
    session-01.md      ← prep + recap in same file
    session-02.md
  loot.md
  rules-house.md       ← home-rules the table agreed on
```
