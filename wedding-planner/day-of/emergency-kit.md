# 🧰 Emergency Kit

The wedding-day "go bag". Kept by the day-of coordinator (or maid of honor / best man if there isn't one). Not by the couple.

## Personal repair

- [ ] Safety pins (multiple sizes) — for ANY clothing emergency
- [ ] Sewing kit (white + black thread minimum)
- [ ] Double-sided fashion tape (for necklines, hems, anything)
- [ ] Static-cling spray
- [ ] Lint roller
- [ ] Stain remover stick (Tide-to-Go or equivalent)
- [ ] Wet wipes — for the dress, for hands, for everything
- [ ] Tissues
- [ ] Deodorant (travel-size, unscented)
- [ ] Bobby pins, hair pins, hair elastics
- [ ] Extra hairspray
- [ ] Bobby pins again (you will need more than you think)

## Health + body

- [ ] Pain relievers (ibuprofen + acetaminophen)
- [ ] Antacids
- [ ] Anti-diarrheal (truly — wedding-day stress + buffet food is real)
- [ ] Bandaids (regular + heel-blister type)
- [ ] Moleskin for shoe blisters
- [ ] Floss
- [ ] Mints (no gum — gum on the dance floor is a hazard)
- [ ] Eye drops
- [ ] Allergy medication (if anyone in the party needs it)
- [ ] Tampons / pads (always — even if "not the time")
- [ ] Inhaler (if any party member uses one)

## Beauty touch-ups

- [ ] Lipstick / lip gloss in the bride's color
- [ ] Powder compact for shine
- [ ] Concealer
- [ ] Q-tips
- [ ] Cotton balls
- [ ] Makeup remover wipes (for fixes)
- [ ] Nail file + clear polish (for chips)

## Logistics

- [ ] Phone chargers (multiple cable types)
- [ ] Portable battery pack (charged)
- [ ] Pen + paper
- [ ] Tape (clear + masking)
- [ ] Scissors
- [ ] Small flashlight
- [ ] Cash in small bills (tips, last-minute purchases)
- [ ] Copy of marriage license
- [ ] Copy of all vendor contracts (PDFs on a thumb drive + on a phone)
- [ ] Day-of contact list (printed)

## Food + drink

- [ ] Snacks (granola bars, almonds, things that don't melt or stain)
- [ ] Water bottles (the bride and groom will not eat or drink unless someone hands it to them)
- [ ] Straws (drinking with lipstick)

## Weather-specific

For an outdoor wedding, add:
- [ ] Umbrellas (multiple — clear or matching the palette)
- [ ] Sunscreen
- [ ] Bug spray
- [ ] Hand fans / parasols if hot
- [ ] Hand warmers if cold
- [ ] Pashminas / wraps for guests if cool

## Where the kit lives

In one labeled bag. Given to the designated coordinator at the rehearsal. Stays with the bride/getting-ready area through the morning, then moves to a known location at the venue (often the bridal suite or vendor green-room) for the rest of the day.

The couple does not carry it. The couple does not manage it. The couple does not even know where it is during the day. That's by design.
