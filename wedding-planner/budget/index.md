# 💰 Budget

The hardest part of wedding budgeting is **honesty**. The first vendor quote IS reality. Adjust the budget the same day, not "next week."

## The standard breakdown

> Use these as defaults. Adjust based on the vision page (smaller guest count = less venue, more flowers vs less; intimate vs large; etc.).

| Category | % of total |
|---|---|
| Venue + catering | ~50% |
| Photography + videography | ~12% |
| Attire (both sides + alterations) | ~10% |
| Flowers + décor | ~10% |
| Music (DJ or band) | ~8% |
| Stationery + signage | ~3% |
| Cake / dessert | ~2% |
| Hair + makeup | ~2% |
| Transportation | ~1% |
| Officiant | ~1% |
| Marriage license | trivial |
| Tips for vendors | ~3% (factor in!) |
| Buffer for surprises | ~5–10% |

## Most common budget surprises

These get missed in early budgets and cause overruns:

- **Vendor tips** — 10–20% on services, often $500–2000 total. Factor in.
- **Tax + service charges** — venue/catering often quote pre-tax + pre-service. Real number is +25–30%.
- **Alterations** — $500–1500 on top of dress price.
- **Day-of stationery** — programs, menus, place cards, signage. Easy $400–800.
- **Hair + makeup for the wedding party** — if you're paying. Bridesmaids' look can run $150–300/each.
- **Welcome bags** for hotel blocks — $10–25/room × 30 rooms = real money.
- **Postage** — invitations + RSVPs + thank-you notes. $100–300.
- **Trial costs** — hair, makeup, cake tasting if not included.
- **Rehearsal dinner** — usually paid by groom's parents traditionally; verify before assuming.
- **After-party** — if having one, the venue rarely covers it.

## Save money where it doesn't show

Couples regret overspending in this order: flowers, stationery, favors, transportation. Couples *don't* regret spending on: photographer, food, an open bar, a great DJ or band.

Reallocate accordingly.
