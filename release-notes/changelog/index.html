v2.4.1
Released May 21, 2026
Latest 2 features 2 improvements 1 fix
◆ Features
F
Passkey authentication (WebAuthn)
You can now sign in using Face ID, Touch ID, or a hardware security key — no password required. Enroll a passkey from Security Settings → Authentication Methods. Works across Chrome 108+, Safari 16+, and Firefox 119+.
F
SSO: IdP metadata auto-import for Okta and Azure AD
Configuring SSO for Okta or Azure AD now takes under 2 minutes — paste your IdP metadata URL and we handle the rest. No manual XML editing required. Available on Pro and Enterprise plans.
⇧ Improvements
I
API rate limiting per workspace
Rate limits now apply per workspace rather than per API key, so reaching a limit in one workspace no longer affects others. Your current limits are visible in API Settings, and workspace admins can request an increase.
I
Up to 62% faster events dashboard
The main events table now loads significantly faster for workspaces with large event histories. No action required — the improvement applies automatically.
✓ Fixes
✓
Unexpected sign-outs after ~1 hour
Fixed an issue where users were occasionally logged out unexpectedly after approximately one hour of inactivity in single-page-app sessions. Sessions now renew correctly in the background.
